# **App Name**: Cortex Fit

## Core Features:

- AI Diet Planner: AI-powered diet plan generation based on user's body stats, fitness goals, and preferences, acting as a planning tool.
- User Profile Input: User input form to gather body stats, fitness goals, and dietary preferences.
- AI Chat Coach: AI chat interface for 24/7 coaching, answering questions and providing support and guidance. Generative AI is used as a conversational tool.
- Diet Plan Display: Clear display of the generated diet plan with meal suggestions, nutritional information, and recommended serving sizes.
- How It Works Section: A 'How It Works' section, briefly describing the process from user input to diet plan generation.
- Footer Section: Footer section with copyright information and a 'Powered by DecoAI' attribution.

## Style Guidelines:

- Primary color: A vibrant, slightly muted purple (#A259FF) to suggest sophisticated technology but also well-being.
- Background color: Clean, desaturated off-white (#F5F3FF) to provide a light and modern feel.
- Accent color: Analogous light blue (#59B1FF), for highlights and call to action buttons.
- Clean and modern font to ensure readability.
- Minimalist, line-style icons for key features and navigation.
- A clean and intuitive layout with a focus on user experience.
